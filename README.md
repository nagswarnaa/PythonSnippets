# Important Python Snippets For Competetive Coding 

### Type-Conversions, Assignments

```python
n=1 
print(n)

#re-assigning
n="123"

#Converting to Integer
n=int(n)
print(type(n))


#Converting to Floating Point
n="1.23"
n=float(n)
print(type(n))

#Converting to Floating Point to Integer
n=1.23
n=int(n)
print(n)

#Multi assign 
x,y=1,2

#Unpacking
x,y=[1,2]

n=1
n+=1
print(1)
```
